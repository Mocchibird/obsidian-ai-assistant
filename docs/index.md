> **About this fork**: This is a free, local-first fork of Copilot for Obsidian. Every feature — including the autonomous agent, web search, image understanding, and more — is unlocked for everyone. There is no paid tier, account, or license key, and nothing is ever sent to a cloud middleman: your prompts and notes go only to the model provider you choose (or stay fully local). See the [project README](../README.md) for the full fork notice and privacy details.

## Quick Start

1. Install the plugin. It is **not** in the Obsidian Community Plugins store — install it via BRAT, a manual release download, or by building from source. See the [README install methods](../README.md#install) for step-by-step instructions.
2. Add an API key in Settings → Copilot → Basic → Set Keys (or point Copilot at a local model like Ollama or LM Studio — no key needed)
3. Open the chat panel with the robot icon in the left ribbon
4. Start chatting!

For a full walkthrough, see [Getting Started](getting-started.md).
